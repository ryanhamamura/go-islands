What is Islands Architecture?

Islands architecture is a pattern where most of the page is static HTML rendered by the server, with interactive "islands" of React components.

This approach allows for better performance and SEO compared to traditional SPAs.

Better Performance

Only load JavaScript for components that need interactivity

Improved SEO

Pages are server-rendered for better indexing and initial load

Progressive Enhancement

Core content works without JavaScript, enhanced with interactivity
